# Business Logic Patterns

A practitioner's guide to categorising, implementing, composing, and managing business logic patterns in enterprise software.

Built with [mdBook](https://rust-lang.github.io/mdBook/).

## Local development

```bash
mdbook serve
```

Then open <http://localhost:3000>.

## Build

```bash
mdbook build
```

The generated site is written to the `book/` directory.

## Published site

The book is automatically built and deployed to GitHub Pages on every push to `main`.
